# Cook4Me

How to launch the application ?

| Application | Path                       | Command                                         |
| ----------- | -------------------------- | ----------------------------------------------- |
| OpenId      | api\src\Cook4Me.Api.OpenId | dotnet run -f net46 --server.urls=http://*:5001 |
| API         | api\src\Cook4Me.Api.Host   | dotnet run -f net46 --server.urls=http://*:5000 |
