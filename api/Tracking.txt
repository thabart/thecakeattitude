CLIENT : Choisir moyen de transport (recevoir colis) => confirmer l'adresse de l'envoie => choisir entre UPS ou BPOST => si point de dépot est affiché alors montrer la carte => résumé de l'achat (afficher le prix + un certain montant variable selon l'envoie)
Lorsqu'on clique sur UPS alors montrer les frais d'envoie : choisir parmis les services => appeler la méthode : https://wwwcie.ups.com/rest/Rate
Lorsqu'on clique sur BPOST alors montrer les frais d'envoie : choisir parmis les services => appeler la méthode : https://parcel.bpost.be/fr/select-weight
VENDEUR : Reçoit la demande => afficher les préférences de réception de la commande (BPOST) OU UPS ou HAND TO HAND => Si BPOST alors afficher le site / Si UPS alors afficher le site => Saisir le prix d'envoie => et confirmer la commande (un email sera ensuite envoyé au client)
